## Project: чатор (Chator)

Uggan wants to build a chat app for Russian teens who are being pushed out of Одноклассники (Odnoklassniki).

**Plan:**
- Name: "чатор" (Chator)
- Logo: Sent via nopaste (PNG image at https://nopaste.net/chator)
- Tech stack idea:
  - Spacebar server (Discord alternative, self-hostable, Discord-compatible)
  - Fermi client (Spacebar client in TS/HTML/CSS)
  - Make Fermi mobile responsive
  - Deploy on Render (free tier, needs ping every 5 min to stay awake)

**Research needed:**
- Spacebar: https://github.com/spacebarchat/spacebarchat
- Fermi: https://github.com/MathMan05/Fermi
- Render free tier limitations

**Notes:**
- This is ambitious - Spacebar is a full Discord reimplementation
- May need PostgreSQL, Redis
- Render free tier has resource limits
- Consider if lighter alternatives exist (Rocket.Chat, Mattermost, or custom simple solution)
